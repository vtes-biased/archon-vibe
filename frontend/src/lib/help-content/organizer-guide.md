# Organizer Guide

This guide walks you through running a tournament with Archon, from creation to final results.

## Creating a Tournament

Go to **Tournaments** and click **+ New Tournament**. Fill in:

- **Name**: The tournament name players will see
- **Format**: Constructed, Limited, or Draft
- **Rank**: The VEKN ranking tier
- **Date & Time**: Start and finish times with timezone
- **Location**: Country, city, venue, and address (or mark as Online)
- **Description**: Optional details (supports Markdown formatting)

## Configuration

In the **Config** tab, you can set:

- **Standings visibility**: Who can see standings during the tournament
- **Decklists visibility**: Who can see submitted decklists
- **Decklist required**: Whether players must submit a deck to check in
- **Allow proxies**: Whether proxy cards are permitted
- **Multideck**: Whether players can submit multiple decks
- **League**: Link the tournament to an active league

## Managing Registration

1. From the **Overview** tab, click **Open Registration**
2. Players can self-register from the tournament page
3. You can manually add players from the **Players** tab using the **Add Player** button
4. When ready, click **Close Registration & Start Check-in**

## Running Rounds

### Check-in Phase

Before each round, players must check in:

- Display the **QR Code** (from Overview) at the venue for players to scan
- Or manually check in players from the player list
- Click **Check All In** to check in all registered players at once

### Starting a Round

Click **Start Round** from the Overview tab. Archon will automatically seat players at tables following VEKN tournament rules, optimizing for fair seating.

### During a Round

- View table assignments in the **Rounds** tab
- Enter scores for each table as they finish
- Use **Alter seating** if you need to move players between tables
- A round ends when all tables are scored — click **End Round**

## Running Finals

After enough rounds (at least 2 with 5+ players), you can start finals:

1. Resolve any ties in the top 5 standings (set toss values if needed)
2. Click **Start Finals** from the Overview tab
3. The top 5 players are seated at the finals table
4. Enter scores in the **Finals** tab
5. Click **Finish Finals** to determine the winner

## Finishing the Tournament

Click **Finish Tournament** from the Overview tab. This locks results and generates final standings and ratings.

## Offline Mode

If you're at a venue without reliable internet:

1. Open the tournament page while online to sync all data
2. Click **Go Offline** from the Overview tab
3. Run the tournament normally — all changes are stored locally
4. When internet is available, click **Go Back Online** to sync everything

> **TIP**: Test offline mode before your tournament day to make sure everything works smoothly.
